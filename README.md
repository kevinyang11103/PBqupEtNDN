# 基于Django的食堂外卖系统 python1281

## 项目概述


## 技术栈

- **后端框架**：Django
- **编程语言**：Python 3.7/3.8
- **数据库**：MySQL 5.7 或更高版本
- **数据库工具**：Navicat 11（推荐）
- **开发环境**：PyCharm/VS Code
- **前端框架**：Vue.js

## 功能模块

- 用户注册与登录
- 菜品浏览与搜索
- 购物车管理
- 订单提交与跟踪
- 用户评论与反馈

## 系统角色

- 用户：浏览菜品、下单购买、评价反馈
- 商家：菜品管理、订单处理、营业统计
- 管理员：系统管理、用户管理、订单管理

## 运行环境

- Python 3.7/3.8
- MySQL 5.7 或更高版本
- Navicat 11（推荐）
- 适用于主流浏览器的前端Vue.js框架

## 部署步骤

1. 环境准备：安装Python、MySQL、Navicat
2. 克隆源代码到本地
3. 安装依赖：使用pip安装项目所需库
4. 数据库配置：根据实际环境配置数据库连接
5. 运行Django项目：`python manage.py runserver`
6. 打包前端Vue.js代码，放置于Django项目静态文件目录下

## 目录结构

```
项目名称/
│
├── app1/
├── app2/
│
├── static/            # 静态文件目录，存放前端Vue.js打包文件
│
├── manage.py
├── requirements.txt   # 依赖列表
│
└── ...
```
